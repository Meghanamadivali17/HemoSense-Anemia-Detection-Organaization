# HemoSense: AI-Powered Anemia Detection System

### Project Domain
AI/ML & Data Science Internship Project (SmartBridge & APSCHE)

### How to Run Locally:
1. Install dependencies: `pip install -r requirements.txt`
2. Train ML Model: `python train_model.py`
3. Start Flask Web Server: `python app.py`
4. Open browser and go to: `http://127.0.0`
